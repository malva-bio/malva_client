Malva Client
============

Python client for the `Malva <https://malva.bio>`_ genomic search platform,
enabling programmatic access to harmonized single-cell and spatial
transcriptomics data.

Key Features
------------

**Search** -- Gene symbols, natural language, DNA sequences, and batch queries.

**Coverage** -- Genomic region and sequence coverage across cell types, filterable by organ, study, and disease.

**Dataset Discovery** -- Browse datasets, studies, samples, and harmonized metadata.
