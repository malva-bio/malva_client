Installation
============

Requires **Python 3.8+**.

.. grid:: 2

   .. grid-item-card:: From PyPI

      .. code-block:: bash

         pip install malva_client

   .. grid-item-card:: From Source

      .. code-block:: bash

         git clone https://github.com/malva-bio/malva_client
         cd malva_client
         pip install -e .

Verify Installation
-------------------

.. code-block:: bash

   python -c "import malva_client; print(malva_client.__version__)"
